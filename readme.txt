install gorilla websocket in the working directory if not installed
run main.go - this starts the server

To open the client
- open a browser and type http://localhost:8080

to run the client on another computer
- find the IPv4 Address of the machine running main.go
- make sure to have both computers connected to the same network
- type http://192.168.43.121:8080/ - where xxx.xxx.xxx.xxx is the given ip address.
